# デジタル昭和横丁（DEMO版）

本リポジトリは、HEW進級制作展（2026）にて展示した  
チーム制作作品 「デジタル昭和横丁」 の DEMO バージョンです。

## 概要

レトロな商品や雑貨を中心に販売するオンラインストアです。  
世界の駄菓子を扱うECサイトにガチャ・じゃんけん機能を組み込み、購入を楽しみに変える体験型ECサイトです。

### 機能

- **商品表示**：トップページで「今月のおすすめ」と「商品一覧」を表示
- **商品詳細**：商品詳細ページから数量を指定してカート追加
- **ユーザー機能**：新規登録 / ログイン / ログアウト
- **マイページ**：ユーザー情報、注文履歴、アイコン変更
- **カート機能**：追加・数量更新・削除時に在庫予約と在庫返却を同期
- **注文機能**：チェックアウト、引き券選択、注文確定、完了ページ表示
- **メール送信**：Flask-Mail による注文完了メール送信（※DEMO版にはなし）
- **ミニゲーム**：ガチャ / じゃんけんの2コンテンツ
- **券システム**：`わくわく券`（ゲーム消費）と `5%引き券 / 10%引き券`（割引適用）
- **管理機能**：在庫管理、商品追加、商品編集、商品削除
- **情報ページ**：配送、返金、お問い合わせ、プライバシーポリシー、利用規約、サイト概要

## 制作背景

### 課題

- 一般的な EC サイトは購入完了後の継続導線が弱くなりやすい
- 長らく訪問のきっかけが少なく、体験が単発で終わりやすい

### 解決方針

- 購入と報酬を連動させる「券ベースのゲーム導線」を導入
- 既存の購買導線を保ちながら、軽量なゲーム体験で再訪を促進
